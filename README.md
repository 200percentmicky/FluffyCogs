Fun cogs that I've made for Red V3.

# Installation

To install:

> [p]repo add FluffyCogs https://github.com/zephyrkul/FluffyCogs

> [p]cog list FluffyCogs

> [p]cog install FluffyCogs <cog_name>

> [p]load <cog_name>

> [p]help <Cog_Name>
